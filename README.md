# MCMDIS

This is the official implementation of the paper "Stable Separation of Non-Additive Interactions under Neural Approximation".

To run the experiment, first download our benchmark from [https://drive.google.com/file/d/1RMPIcD23Gzl2Dd4H9EkmGtP1MkGLjz4b/view?usp=share_link] and place all subfolders contained within in the same directory as `mcmdis.py`. Then run the following (with Anaconda):

```
conda create -y -n mcmdis python=3.10
conda activate mcmdis
pip install -r requirements.txt
```
